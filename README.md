# GoDoList
CLI Todo List in GO Lang.

## Requirements
[Go Lang](https://go.dev/)

## Installation

### Stable Version

Download The GoDoList from the `Source Repo`.

```bash
git clone https://github.com/GreatRSingh/GoDoList
```

### GoDoList Basic Usage

```bash
cd GoDoList
./todo -add task1
./todo -complete=1
./todo -del=1
./todo -list
```
